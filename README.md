# Tetris

Tetris on klassinen peli, jossa pinotaan eri muotoisia palikoita pelikentälle, tavoitteena saada virheettömiä rivejä aikaiseksi. Peli on vielä vaiheessa.

## Python-versio

Sovellus on testattu Python-versiolla 3.8.
